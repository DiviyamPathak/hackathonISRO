Description:
The data provided in this folder contains the processed Chennai Doppler Weather Radar reflectivity and radial velocity informaton along with its geographic locations
in standard Netcdf format. 

Size: Each Netcdf file is of 520 KB size

Built with: The Netcdf files are created usinging MATLAB software
